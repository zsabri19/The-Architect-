## Typography

### Font Families

**Playfair Display** (Google Fonts)
- Weights: 400, 500, 600, 700 + italics
- Usage: Headlines, hero text, display typography
- Style: Elegant serif, human touch

**Inter** (System + Google Fonts)
- Weights: 300, 400, 500, 600
- Usage: Body text, UI, navigation, buttons
- Style: Modern sans-serif, highly readable

### Fluid Type Scale

## Layout

### Container
- **Max width:** 1440px
- **Padding X:** 1.5rem (mobile) → 4rem (desktop)
- **Section gap:** 5rem (mobile) → 9rem (desktop)

### Spacing Scale
- Section padding: `clamp(5rem, 10vw, 9rem)`
- Content gap: 24px - 48px
- Card padding: 24px - 48px

### Border Radius
- Small: 6px
- Medium: 12px
- Large: 20px
- XL: 32px

---

## Components

### Buttons

**Primary Button**
- Background: Wisdom Gold (`#c9a227`)
- Text: Void (`#0a0a0f`), uppercase
- Padding: 16px 32px
- Border-radius: 8px
- Hover: Shine (`#e8c547`), lift -3px
- Shadow: `0 0 40px rgba(201, 162, 39, 0.15)`

**Ghost Button**
- Background: Transparent
- Border: 1px solid Dawn
- Text: Dawn
- Hover: Background Dawn, text Void

### Cards

**Standard Card**
- Background: Shadow (`#12121a`)
- Border-radius: 12px - 20px
- Padding: 24px - 48px
- Shadow: `0 2px 24px rgba(0, 0, 0, 0.4)`
- Hover: Lift 8px, shadow intensify

**Feature Card**
- Background: Gradient or Shadow
- Border-left: 4px solid Wisdom Gold
- Border-radius: 12px
- Padding: 32px

## Visual Effects

### Shadows
- Gold glow: `0 0 40px rgba(201, 162, 39, 0.15)`
- Card shadow: `0 2px 24px rgba(0, 0, 0, 0.4)`
- Lift shadow: `0 8px 48px rgba(0, 0, 0, 0.5)`

### Transitions
- Fast: 180ms
- Medium: 320ms
- Slow: 550ms
- Easing out: `cubic-bezier(0.22, 1, 0.36, 1)`
- Easing in: `cubic-bezier(0.64, 0, 0.78, 0)`

### Effects
- Hover lift: `transform: translateY(-3px)`
- Scroll reveals: Fade in + slide up
- Gold glow on focus/active states

## Voice & Tone

### Brand Voice
- **Clear:** Zero ambiguity, no fluff
- **Confident:** Expert authority, bold statements
- **Human:** Warm, accessible, not corporate
- **Precise:** Every word has purpose

### Writing Principles
1. Lead with the insight, not the setup
2. Use active voice
3. Avoid jargon unless necessary
4. Every message needs a clear next step
